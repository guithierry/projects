## Projects

#### Backend

-   Java
-   Spring Boot
-   MySQL

#### Frontend

-   React.js
-   [React-dnd](https://react-dnd.github.io/react-dnd/about)
-   Bootstrap

## Run

```bash
git clone https://github.com/guithierry/projects.git
cd projects
```

#### Backend

```bash
cd backend

mvn spring:boot-run
```

#### Frontend

```bash
cd frontend

yarn

yarn dev

```
